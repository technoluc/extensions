## Custom site search
`brave://settings/searchEngines`

|Search Engine|Shortcut|URL with %s (use %S for space instead of "+")|
|:-:|:-:|:-:|
|GitHub|gh|https://github.com/search?q=%S&type=repositories|
|Kamer van Koophandel|kvk|https://tinyurl.com/kvkuitgebreid|
